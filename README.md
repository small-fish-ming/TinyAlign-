# TinyAlign：超轻量级大模型全链路训练与人类偏好对齐

TinyAlign 是一个**超轻量级、全流程可复现**的大语言模型训练框架，覆盖预训练、指令微调（SFT）、人类偏好对齐（PPO / DPO / GRPO）、自动化评测（LLM-as-Judge）完整链路，旨在低资源条件下实现高效、稳定、高质量的模型训练与对齐。

---

## 项目简介
本项目设计并实现 **TinyAlign 超轻量级大语言模型全链路训练体系**，从基座模型预训练到人类偏好对齐，再到自动化效果评测，形成完整闭环。
项目以轻量化、可复现、工业级可用为目标，适合学习、研究与快速迭代。

---

## 主要内容

### 1. 预训练（Pre-training）
- 基于**匠数大模型数据集**中文部分，清洗构建约 **6GB 高质量语料**
- 采用 **MoE 架构** 完成轻量级基座模型训练
- 构建基础语义理解、知识记忆与文本生成能力

### 2. 指令微调（SFT）
- 基于 **Qwen 开源 Magpie-SFT 高质量对话数据集（1.2GB）**
- 构建标准化中文指令 + 多轮对话数据体系
- 显著提升模型：
  - 指令遵循能力
  - 对话交互能力
  - 多轮语义理解能力

### 3. 人类偏好对齐（Preference Alignment）
项目实现 **三种主流对齐算法**，支持低资源训练：

#### ✅ PPO (Proximal Policy Optimization)
- 基于强化学习的经典偏好对齐框架
- 使用 **InternLM2 奖励模型** 提供语义奖励信号
- 加入 **KL 散度约束**，保证训练稳定，避免模型偏离基座能力

#### ✅ DPO (Direct Preference Optimization)
- 无需显式奖励模型，简化对齐流程
- 基于 **Qwen 开源 Magpie-DPO 数据集（0.9GB）**
- 训练稳定、速度快、对齐效果优秀

#### ✅ GRPO (Group Relative Preference Optimization)
- 分组偏好强化学习算法
- 同样使用 **InternLM2 奖励模型**
- 在弱偏好数据、低资源条件下表现更优

### 4. 自动化评测（LLM-as-Judge）
- 构建 **10 道中文权威测评问题**
- 使用 **DeepSeek** 作为自动评判模型（LLM-as-Judge）
- 从三大维度量化评估模型效果：
  1. 事实准确性
  2. 内容完整性
  3. 逻辑严谨性
- 自动对比不同训练/对齐策略的效果增益

## 项目特点
- 🔥 **超轻量级**：低资源可跑，适合个人与实验室环境
- 📌 **全链路覆盖**：预训练 → SFT → 偏好对齐 → 自动化评测
- 🧩 **多算法支持**：PPO / DPO / GRPO 三种对齐方式
- 📊 **自动化评估**：LLM-as-Judge 量化打分，结果可复现
- 🧠 **中文优化**：全部基于高质量中文数据集训练

## 使用说明
本仓库仅保留**核心源码**，不含数据集与模型权重。
你可以根据需要运行：
- 预训练
- 指令微调 SFT
- 偏好对齐 PPO / DPO / GRPO
- LLM-as-Judge 自动化评测
